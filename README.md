# 🖌Commands | 指令
```
/shulkerview reload - 重新載入插件設定檔 

/shulkerview on - 開啟界伏盒預覽功能

/shulkerview off - 關閉界伏盒預覽功能
```

# 🔓Permissions | 權限
```
command.shulkerview.reload - 使用 reload 的權限
command.shulkerview.switch - 使用 on/off 的權限
```
